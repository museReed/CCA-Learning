| 项目 | 细节 |
|------|--------|
| 考试领域 | D2 — 工具设计与 MCP 整合 (18%) |
| 任务陈述 | 2.1 (MCP 设定与组态) |

---

## 一句话摘要

本课提供实作 MCP 课程的起始项目文件（cli_project.zip），建立你在后续课程中用來建构 MCP 服务器和客户端的本地开发环境。

---

## 设定步驟

1. **下載** `cli_project.zip` 起始范本（或 `cli_project_COMPLETE.zip` 完成版）
2. **解压缩**到你的工作目录
3. **安装依賴项** — 项目使用 Python 搭配 MCP SDK
4. **验证**项目结构包含预期的服务器和客户端骨架

起始项目提供预先设定的环境，让你能專注於学習 MCP 概念而非样板设定。

---

## CCA 考试关聯

- **D2**：理解本地 MCP 开发设定是基础。考试假设你熟悉 MCP 服务器和客户端如何设定和连接。
- 这是实作设定课 — 考试不会直接问设定步驟，但项目结构有助於理解 MCP 架构。
