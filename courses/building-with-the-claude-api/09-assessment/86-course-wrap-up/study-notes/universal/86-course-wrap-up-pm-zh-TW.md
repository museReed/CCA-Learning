## 一句話摘要

本課提供課程所有主要主題的鳥瞰圖，並指出自學優先順序最高的後續領域：代理編排、代理監控、Agentic RAG 和工具評估。

---

## 課程主題一覽

| 主題 | 商業影響 | 關鍵要點 |
|------|----------|---------|
| **模型選擇** | 成本 vs 能力取捨 | Haiku 注重速度/成本，Sonnet 注重智能 |
| **API 參數** | 控制輸出品質 | Temperature、stop sequences、prefilling |
| **提示詞評估** | 生產可靠性 | 最重要的實踐 — 不可跳過 |
| **提示工程** | 回應品質 | 清晰度是最主要的槓桿 |
| **工具使用** | 能力擴展 | 讓 Claude 與外部系統互動 |
| **工作流程 vs 代理** | 架構決策 | 優先使用工作流程 — 更高準確度、更可預測 |

### 最大的要點

**提示詞評估是任何 AI 產品最重要的實踐。** 在測試中運作 10 次的提示詞可能在大規模時失敗。評估不需要複雜的框架 — Claude 可以幫助建構它們。

### 工作流程優先於代理

在規劃產品功能時：**預設使用工作流程**，而非代理。代理靈活但不太可預測。工作流程為結構化任務提供一致、可衡量的結果。
