## 一句話摘要

當 skill 能在團隊或組織中共享時，它們會變得更有價值。本影片涵蓋三種主要的發布方式 — 儲存庫 commit、plugin 和企業管理設定 — 並說明如何設定自訂 subagent 來使用 skill。

---

## 為什麼 PM 需要知道

### 學完本課你將理解

- 透過將 skill commit 到 Git 儲存庫來與團隊共享
- 透過 plugin 和市場跨專案發布 skill
- 使用企業管理設定在全組織範圍部署 skill
- 設定自訂 subagent 來使用特定的 skill

### 重點摘要（商業影響）

專案 skill 放在 .claude/skills 中，會透過 Git 自動共享 — 任何 clone 該儲存庫的人都會自動取得。
