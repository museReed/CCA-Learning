專案設定 — PM 快速掃描

項目 細節
考試領域 D2 — 工具設計與 MCP 整合 (18%)
任務陳述 2.1 (MCP 設定與組態)
來源 introduction-to-model-context-protocol / 01-mcp-basics / 第 05 課

一句話摘要

本課提供可下載的起始檔案用於實作 MCP 專案，建立整個課程使用的開發環境。

重點

  • 課程提供起始範本（cli_project.zip），讓學員專注於 MCP 概念
  • 也提供完成版（cli_project_COMPLETE.zip）供參考
  • 專案建立本地開發模式：MCP 伺服器 + 客戶端一起運行

CCA 考試關聯

設定課 — 不會直接出考題，但專案結構有助於理解 MCP 架構。

記憶卡

問題 答案
MCP 課程的起始專案包含什麼？ 預先設定的伺服器和客戶端骨架，用於實作 MCP 開發。
為什麼課程使用起始範本？ 讓學員專注於 MCP 概念而非設定樣板。
