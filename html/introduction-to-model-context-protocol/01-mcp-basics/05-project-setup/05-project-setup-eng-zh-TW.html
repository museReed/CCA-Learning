專案設定 — 工程師深度解析

項目 細節
考試領域 D2 — 工具設計與 MCP 整合 (18%)
任務陳述 2.1 (MCP 設定與組態)
來源 introduction-to-model-context-protocol / 01-mcp-basics / 第 05 課

一句話摘要

本課提供實作 MCP 課程的起始專案檔案（cli_project.zip），建立你在後續課程中用來建構 MCP 伺服器和客戶端的本地開發環境。

設定步驟

  1. 下載 cli_project.zip 起始範本（或 cli_project_COMPLETE.zip 完成版）
  2. 解壓縮到你的工作目錄
  3. 安裝依賴項 — 專案使用 Python 搭配 MCP SDK
  4. 驗證專案結構包含預期的伺服器和客戶端骨架

起始專案提供預先設定的環境，讓你能專注於學習 MCP 概念而非樣板設定。

CCA 考試關聯

  • D2：理解本地 MCP 開發設定是基礎。考試假設你熟悉 MCP 伺服器和客戶端如何設定和連接。
  • 這是實作設定課 — 考試不會直接問設定步驟，但專案結構有助於理解 MCP 架構。

記憶卡

問題 答案
MCP 課程的起始專案提供什麼？ 預先設定的環境，包含用於建構 MCP 應用程式的伺服器和客戶端骨架。
專案有哪兩個下載選項？ cli_project.zip（起始版）和 cli_project_COMPLETE.zip（完成版）。
為什麼使用起始範本而不從頭建構？ 為了專注於學習 MCP 概念，而非花時間在樣板設定上。
