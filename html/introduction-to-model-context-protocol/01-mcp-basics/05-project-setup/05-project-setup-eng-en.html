One-Liner

This lesson provides the starter project files (cli_project.zip) for the hands-on MCP course, establishing the local development environment you'll use to build MCP servers and clients throughout the remaining lessons.

Setup Steps

  1. Download the cli_project.zip starter template (or cli_project_COMPLETE.zip for the finished version)
  2. Extract to your working directory
  3. Install dependencies — the project uses Python with the MCP SDK
  4. Verify the project structure includes the expected server and client scaffolding

The starter project gives you a pre-configured environment so you can focus on learning MCP concepts rather than boilerplate setup.

CCA Exam Relevance

  • D2: Understanding the local MCP development setup is foundational. The exam assumes familiarity with how MCP servers and clients are configured and connected.
  • This is a practical setup lesson — exam questions won't ask about the setup steps directly, but the project structure informs your understanding of MCP architecture.
