一句话摘要

本课提供可下載的起始文件用於实作 MCP 项目，建立整个课程使用的开发环境。

重点

  • 课程提供起始范本（cli_project.zip），让学員專注於 MCP 概念
  • 也提供完成版（cli_project_COMPLETE.zip）供参考
  • 项目建立本地开发模式：MCP 服务器 + 客户端一起运行

CCA 考试关聯

设定课 — 不会直接出考题，但项目结构有助於理解 MCP 架构。
