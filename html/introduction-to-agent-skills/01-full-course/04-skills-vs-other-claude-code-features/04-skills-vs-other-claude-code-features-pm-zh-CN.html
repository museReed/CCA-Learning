Skills vs. other Claude Code features — PM 观点

项目 细节
考试范围 D3 — Claude Code 设定与工作流程 (20%)
任务陈述 3.1 (CLAUDE.md)、3.3 (自订指令/skill)、3.5 (权限模型)
课程来源 introduction-to-agent-skills / 第 04 课

一句话摘要

Claude Code 提供多种自订选项，选错可能导致不必要的复杂度。本影片拆解何时应使用 skill，而非 CLAUDE.md、subagent、hook 和 MCP server。

为什么 PM 需要知道

学完本课你将理解

  • 比较 skill 与 CLAUDE.md、subagent、hook 和 MCP server 的差异
  • 針对特定使用情境选择正确的 Claude Code 自订功能
  • 设计一个结合多种功能的互补配置

重点摘要（商業影响）

CLAUDE.md 会载入每一次对话，最适合用于「始终开启」的專案标准。skill 则按需载入，最适合用于特定任务的專業知识。

PRD 檢查清单

  • [ ] 團隊是否理解 skill 与其他 Claude Code 功能的差异？
  • [ ] 相关功能是否已记录在文件中？
  • [ ] 是否已考慮邊界情況？
