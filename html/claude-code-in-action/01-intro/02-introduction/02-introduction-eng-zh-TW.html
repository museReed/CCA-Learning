一句話理解

Anthropic 的 Steven Greider 介紹這門 4 部分的課程：什麼是 coding assistant、Claude Code 的獨特之處、實際動手做專案、以及如何在自己的專案中最大化 Claude Code 的效用。

課程路線圖

部分 主題 對應考試 Domain
1 什麼是 coding assistant D1（Agentic Architecture）、D2（Tool Design）
2 Claude Code 有什麼特別的 D3（Claude Code Config & Workflows）
3 在專案中實際操作 Claude Code D3（Config）、D4（Testing & Eval）
4 如何在你自己的專案中最大化 Claude Code D3（Workflows）、D5（Reliability）

你會學到什麼

  • 理解 coding assistant 這個類別，以及 Claude Code 在其中的定位
  • 深入了解 Claude Code 的差異化功能（tool use、agentic loop、configuration）
  • 以專案為基礎的實作演練——不只是理論
  • 可以套用到實際工程 workflow 的最佳化策略

考試重點

💡 這堂課對考試的意義

這是課程總覽，本身沒有可考的內容。但它建立了整門課的 mental map。Parts 1-2 對應 D1-D3（architecture 和 configuration），Parts 3-4 連結到 D4-D5（testing、evaluation、reliability）。

講師是 Anthropic 的 Steven Greider——知道資訊來源的權威性，有助於理解後續課程中的指導方向。
