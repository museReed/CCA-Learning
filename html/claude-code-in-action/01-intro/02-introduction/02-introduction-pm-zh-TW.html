TL;DR

Anthropic 的 Steven Greider 介紹這門 4 部分的課程，從「什麼是 coding assistant」到「如何在專案中最大化 Claude Code」。對 PM 來說，這門課提供了撰寫更好的 PRD、評估 AI 輔助開發 workflow、以及與工程團隊有效溝通所需的技術詞彙和心智模型。

課程路線圖 — PM 視角

部分 主題 PM 為什麼要關心
1 什麼是 coding assistant 了解你正在使用/圍繞的產品類別
2 Claude Code 有什麼特別的 規劃功能範圍時要知道工具的能力與限制
3 實際操作 Claude Code 看到工程師每天使用的實際開發體驗
4 最大化 Claude Code 學習可以納入團隊最佳實踐的優化模式

你會學到什麼

  • 理解 AI coding assistant 在開發生命週期中的定位
  • 釐清 Claude Code 相比其他工具的獨特價值主張
  • 實際接觸工程師如何與 Claude Code 互動——有助於撰寫合理的 acceptance criteria
  • 團隊層級的導入與 workflow 優化策略

考試重點

💡 PM 重點提醒

這堂課是課程總覽，沒有直接可考的內容。把它當作 mental map：Parts 1-2 建立概念理解（D1-D3），Parts 3-4 建立實作技能（D3-D5）。考試兩者都考——既要知道功能做什麼，也要知道什麼時候該用。

講師是 Anthropic 的 Steven Greider。身為 PM，知道這門課直接來自產品開發者，代表這裡的指導反映的是官方設計意圖，而不是社群的變通方法。
