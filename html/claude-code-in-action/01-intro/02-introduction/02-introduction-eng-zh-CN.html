一句话理解

Anthropic 的 Steven Greider 介绍这门 4 部分的课程：什么是 coding assistant、Claude Code 的独特之处、实际动手做项目、以及如何在自己的项目中最大化 Claude Code 的效用。

课程路线图

部分 主题 对应考试 Domain
1 什么是 coding assistant D1（Agentic Architecture）、D2（Tool Design）
2 Claude Code 有什么特别的 D3（Claude Code Config & Workflows）
3 在项目中实际操作 Claude Code D3（Config）、D4（Testing & Eval）
4 如何在你自己的项目中最大化 Claude Code D3（Workflows）、D5（Reliability）

你会学到什么

  • 理解 coding assistant 这个类别，以及 Claude Code 在其中的定位
  • 深入了解 Claude Code 的差异化功能（tool use、agentic loop、configuration）
  • 以项目为基础的实操演练——不只是理论
  • 可以套用到实际工程 workflow 的优化策略

考试重点

💡 这堂课对考试的意义

这是课程总览，本身没有可考的内容。但它建立了整门课的 mental map。Parts 1-2 对应 D1-D3（architecture 和 configuration），Parts 3-4 连接到 D4-D5（testing、evaluation、reliability）。

讲师是 Anthropic 的 Steven Greider——知道信息来源的权威性，有助于理解后续课程中的指导方向。
